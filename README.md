# AuraGBDB

AuraGBDB is a database-based extension of the AuraGB programming language.

## Features
- Custom compiler (.exe)
- Batch execution support
- Database integration using student.db

## How to Run
1. Download files
2. Run run.bat
3. Use auragb_compiler.exe
